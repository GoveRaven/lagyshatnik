Циклы

  • for (... in ...) - проходит по всем перечислимым свойствам объекта. JavaScript выполнит указанные выражения для каждого отдельного свойства.

  • for(...of...) - создаёт цикл, проходящий по перечислимым объектам, вызывая на каждой итерации функцию с выражениями, которые надо выполнить для получения значения каждого отдельного свойства.

Разница между for...in и for...of: for...in проходит по именам свойств, for...of проходит по значениям свойств:

Глобальные функции

  • obj.toString() - возвращает строку, представляющую объект. На месте obj может быть число или объект.

    Если метод применяется к числу, то в скобках можно указать в какой системе счисления будет переведено число.

  • String(thing) - создаёт строку из thing.

  • eval(string) выполняет JavaScript-код, представленный строкой.

Методы строк

  • charAt(index) - возвращает указанный символ из строки.

    index - целое число от 0 до длины строки минус 1.

  • charCodeAt(index) - возвращает числовое значение Юникода для символа по указанному индексу

  • Split() - разбивает объект String на массив строк путём разделения строки указанной подстрокой.

  • Slice() - извлекает часть строки и возвращает новую строку без изменения оригинальной строки.

  • IndexOf() - служит для поиска. При вызове нужно передать, что искать. Вернётся индекс первого найденного элемента или -1, если ничего не нашлось. работает так же, как с массивами, но можно искать не только буквы в строке, но и подстроки

  • lastIndexOf(searchValue[, fromIndex]) - возвращает индекс последнего вхождения указанного значения в строковый объект String, на котором он был вызван, или -1, если ничего не было найдено. Поиск по строке ведётся от конца к началу.

  • startsWith(searchString, position) - помогает определить, начинается ли строка с символов указанных в скобках, возвращая, соответственно, true или false. searchString - символы, искомые в начале данной строки; position - необязательный параметр. Позиция в строке, с которой начинать поиск searchString; по умолчанию 0.

  • endsWith() - позволяет определить, заканчивается ли строка символами указанными в скобках, возвращая, соответственно, true или false. В скобках указывается то, на что строка должна заканчиваться.

  • replaceAll(pattern, replacement) - возвращает новую строку со всеми совпадениями pattern, который меняется на replacement. pattern может быть строкой или регулярным выражением, и replacement может быть строкой или функция возвращающая каждое совпадение.

  • replace(pattern, replacement) - похож на raplaceAll, разница в том, что этот метод изменяет первый элемент попадающий под pattern, меняя его, и игнорируя остальные

  • toLowerCase() - возвращает значение строки, преобразованное в нижний регистр. Метод не изменяет значение самой строки.

  • toUpperCase() - возвращает значение строки, преобразованное в верхний регистр. Метод не изменяет значение самой строки.

  • includes(searchString, position) - проверяет, содержит ли строка заданную подстроку, и возвращает, соответственно true или false.

    searchString - трока для поиска в данной строке. position - Позиция в строке, с которой начинать поиск строки searchString, по умолчанию 0.

  • str.repeat(count) - конструирует и возвращает новую строку, содержащую указанное количество соединённых вместе копий строки, на которой он был вызван.

    count - Целое число, определяющее число повторений строки во вновь создаваемой и возвращаемой строке.

Методы чисел

  • num.toString(base) - возвращает строковое представление числа num в системе счисления base. base может варьироваться от 2 до 36 (по умолчанию 10).

  • isNaN(value) преобразует value в число и проверяет является ли оно NaN. Есть более строгий метод - Number.isNaN(value) возвращает true только в том случае, если аргумент принадлежит к типу number и является NaN. Во всех остальных случаях возвращает false.

  • isFinity(value) - преобразует аргумент в число и возвращает true, если оно является обычным числом, т.е. не NaN/Infinity/-Infinity. Также имеет более строгий метод - Number.isFinite(value) возвращает true только в том случае, если аргумент принадлежит к типу number и не является NaN/Infinity/-Infinity. Во всех остальных случаях возвращает false.

  • parseInt(str, radix) и parseFloat - Они «читают» число из строки. Если в процессе чтения возникает ошибка, они возвращают полученное до ошибки число. Функция parseInt возвращает целое число, а parseFloat возвращает число с плавающей точкой. Вернут NaN, если не смогли прочитать ни одну цифру

    Функция parseInt() имеет необязательный второй параметр. Он определяет систему счисления, таким образом parseInt может также читать строки с шестнадцатеричными числами, двоичными числами и т.д.

  • toFixed(digits) - форматирует число, используя запись с фиксированной запятой.

    digits - количество цифр после десятичной запятой

Методы массивов

  • unshift(element) - добаляет элементы в начало массива. element - то что добавляется

  • push(element) - добаляет элементы в конец массива. element - то что добавляется.

  • pop() - извлекает элемент массива из конца.

  • shift() - извлекает элемент массива из начала.

  • reverse() - обращает порядок следования элементов массива. Первый элемент массива становится последним, а последний — первым.

  • join() - объединяет все элементы массива в строку.

    Важно: если написать join(), то все объекты массива будут объеденены и перечислены через запятую; если написать join(' ') - массив соединиться слитно без пробелов и каких-либо знаков. Внутри в кавычках скобок можно написать любой знак, который будет добавляться между элементами массива

  • from() - создаёт новый экземпляр Array из массивоподобного (объектов со свойством length и элементами по индексным ключам) или итерируемого объекта (объектов, из которых вы можете достать их элементы, например Map или Set).

  • slice() - возвращает новый массив, содержащий копию части исходного массива.

  • Splice(index, deleteCount, elem1, ..., elemN) - изменяет содержимое массива, удаляя существующие элементы и/или добавляя новые.

    Он начинает с позиции index, удаляет deleteCount элементов и вставляет elem1, ..., elemN на их место. Возвращает массив из удалённых элементов.

  • IndexOf() - служит для поиска. При вызове нужно передать, что искать. Вернётся индекс первого найденного элемента или -1, если ничего не нашлось.

  • filter() - позволяет получить новый массив, отфильтровав элементы с помощью переданной колбэк-функции. Колбэк-функция будет вызвана для каждого элемента массива и по результату функции примет решение включать этот элемент в новый массив или нет.

  • sort() - на месте сортирует элементы массива и возвращает отсортированный массив. Порядок сортировки по умолчанию соответствует порядку кодовых точек Unicode. Копия массива не создаётся - массив сортируется на месте

    array.sort([compareFunction])

    compareFunction - необязательный параметр. Указывает функцию, определяющую порядок сортировки. Если опущен, массив сортируется в соответствии со значениями кодовых точек каждого символа Unicode, полученных путём преобразования каждого элемента в строку.

    Значение compareFunction, для сортировки чисел по возрастанию:

    function compareNumbers(a, b) { return a - b; }

  • includes(searchElement, fromIndex) - определяет, содержит ли массив определённый элемент, возвращая в зависимости от этого true или false.

    searchElement - искомый элемент. fromIndex - позиция в массиве, с которой начинать поиск элемента searchElement.

  • forEach() выполняет указанную функцию один раз для каждого элемента в массиве.

  • map() создаёт новый массив с результатом вызова указанной функции для каждого элемента массива.

  • some() - проверяет, удовлетворяет ли какой-либо элемент массива условию, заданному в передаваемой функции. Возвращает булевое значение

  • every() - проверяет, удовлетворяют ли все элементы массива условию, заданному в передаваемой функции. Возвращает булевое значение

  • reduce() - применяет функцию reducer к каждому элементу массива (слева-направо), возвращая одно результирующее значение. Одно из применений: складывать значения массивов. Последний указываемый параметр после функции - начальное значение при сумме.

  • reduceRight() - делает то же самое, что и метод reduce, то справа-налево

  • fill (value, start, end) - заполняет все элементы массива указанным значение value. start и end означают начальный и конечный индекс

Методы Math

  • Math.random() возвращает псевдослучайное число с плавающей запятой из диапазона [0, 1), то есть, от 0 (включительно) до 1 (но не включая 1), которое затем можно отмасштабировать до нужного диапазона.

  • Math.round() - возвращает число, округлённое к ближайшему целому.

  • Math.ceil() - округление вверх. Округляет аргумент до ближайшего большего целого.

  • Math.floor() - округление вниз. Округляет аргумент до ближайшего меньшего целого.

  • Math.pow(base, exponent) возвращает основание, возведённое в степень показатель, то есть, значение выражения основаниепоказатель.

    base - основание степени; exponent - показатель степени

    Можно заменить знаком **. Слева пишется основание степени, справа - показатель.

Создание элементов в DOM

  • document.createElement(tagname) - создаёт элемент c тем тегом, что указан в аргументе

    строка, указывающая элемент какого типа должен быть создан.

  • parentElement.innerHTML = htmlString - устанавливает или получает HTML разметку дочерних элементов.

    Установка значения innerHTML удаляет всё содержимое элемента и заменяет его на узлы, которые были разобраны как HTML, указанными в строке htmlString.

Перемещение DOM-элементов

  • element.appendChild(child) - добавляет элемент в конец списка дочерних элементов указанного родителя.

    element - родительский элемент

    child - элемент, который вставляется в конец element

  • element.removeChild(child) - удаляет дочерний элемент. Возвращает удалённый элемент, если записать в переменную

  • parentElement.insertBefore(newElement, referenceElement) - добавляет элемент в список дочерних элементов родителя перед указанным элементом.

    parentElement - родитель для нового элемента.

    newElement - элемент для вставки.

    referenceElement - элемент, перед которым будет вставлен newElement.

  • parentNode.replaceChild(newChild, oldChild) - заменяет дочерний элемент на выбранный. Возвращает заменённый элемент.

    newChild элемент на который будет заменён oldChild.

    oldChild элемент который будет заменён.

  • var dupNode = node.cloneNode(deep) - возвращает дубликат узла, из которого этот метод был вызван.

    node - Узел, который будет клонирован.

    dupNode - Новый узел, который будет клоном node

    deep - true, если дети узла должны быть клонированы или false для того, чтобы был клонирован только указанный узел.

Методы объектам

  • hasOwnProperty(prop) - возвращает логическое значение, указывающее, содержит ли объект указанное свойство.

    prop - Имя проверяемого свойства.

Для общего развития

  • Метод Window.prompt() отображает диалоговое окно с необязательным запросом на ввод текста.

    result = window.prompt(message, default);

    result — это строка, содержащая текст, поступивший от пользователя или null.

    message — это строка текста, которая показывается пользователю. Этот параметр является необязательным и может быть пропущен если в окне prompt ничего не показывать.

    default — это строка, содержащая значение по умолчанию, отображаемое в поле ввода текста. Это необязательный параметр. Обратите внимание, что в Internet Explorer 7 и 8, если вы не укажете этот параметр, строка "underfined" будет значением по умолчанию.

  • setTimeOut(function, delay) - Вызов функции или выполнение фрагмента кода после указанной задержки.

    function - функция которая будет выполнена по истечению delay (в милисекундах).

  • clearTimeout(timeoutID) метод отменяет таймаут, ранее установленный вызовом setTimeout().

    timeoutID - Идентификатор таймаута, который вы хотите отменить.
